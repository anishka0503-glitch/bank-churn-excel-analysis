# Bank Churn Analysis (Excel)

This project analyses bank customer churn using Excel.

Tools Used:
- Microsoft Excel
- Pivot Tables
- Basic Data Cleaning

Dataset Insights:
- Total customers: 10000
- Churn rate: ~20%
- High balance inactive customers have higher churn risk

Analysis Performed:
- Tenure vs Churn
- Activity vs Churn Percentage
- Customer Segmentation

Goal:
Identify patterns behind customer churn and generate business insights.
